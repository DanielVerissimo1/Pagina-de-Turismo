**Página de Turismo** é um projeto desenvolvido durante os estudos na **Rocketseat**, com o objetivo de criar uma interface moderna para apresentação de um destino turístico, destacando pontos importantes, imagens e informações relevantes.

---

<img width="400" height="500" alt="preview" src="COLOQUE_PRINT_1_AQUI" />


---

## 🚀 Funcionalidades:

- Apresentação de um destino turístico
- Exibição de pontos turísticos
- Layout moderno e responsivo
- Uso de imagens ilustrativas
- Estrutura semântica com HTML
- Organização visual em seções

---

## 🛠️ Tecnologias utilizadas:

- HTML5  
- CSS3
